# App Gabi Moreno

This is the module corresponding to the app Gabi Moreno.

Download it from [Google Play](https://play.google.com/store/apps/details?id=soy.gabimoreno).

## Stack

It's using:

- MVVM with coroutines
- Clean Architecture
- Jetpack Compose
- Firebase
- Hilt
- Arrow
- Retrofit
- Moshi
- ExoPlayer
- Glide
- Datastore
- MockK
- Kluent
- MockWebServer
- Paparazzi
- ...
